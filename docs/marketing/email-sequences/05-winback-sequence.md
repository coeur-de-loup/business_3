## Sequence Metrics & Optimization

### Target Metrics

| Metric | Target | Industry Avg |
|--------|--------|--------------|
| Open Rate | 35% | 15-25% |
| Click Rate | 4% | 1-3% |
| Reactivation Rate | 4% | 3-5% |
| Feedback Response | 15% | 5-10% |
| Unsubscribe | <1% | 1-2% |

### Segmentation Strategy

**Segment A: Former Power Users (10+ workflows)**
- Emphasize new features
- Offer high-touch re-engagement (call, custom workflows)
- Provide strong incentive (50% off + bonuses)
- Goal: 10% reactivation

**Segment B: Moderate Users (3-9 workflows)**
- Feature highlights
- Moderate incentive (30% off)
- Offer winback consultation
- Goal: 5% reactivation

**Segment C: Light Users (1-2 workflows)**
- Gather feedback priority
- Soft sell (focus on what's new)
- Free resources first, offer second
- Goal: 2% reactivation

### A/B Test Ideas

**Test 1: Email 1 Subject Line**
- A: "We miss you, {{first_name}}! ü•∫"
- B: "It's been 30 days... what's new?"
- C: "New features you've missed"

**Test 2: Email 3 Incentive**
- A: 50% off first month
- B: First month free
- C: No discount, emphasize features instead

**Test 3: Email 2 Approach**
- A: Feedback request (current)
- B: "What went wrong?" survey
- C: "Can we fix it?" direct offer

### Optimization Checklist

- [ ] Track reactivation by original usage level
- [ ] Analyze feedback themes (why people leave)
- [ ] Monitor which new features bring users back
- [ ] Test different timing (30 vs. 45 vs. 60 days)
- [ ] Create "most requested" features based on feedback
