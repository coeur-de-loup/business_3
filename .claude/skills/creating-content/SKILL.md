# Creating Content

Delegates content creation to the content-writer subagent.

## When to Use

- Writing landing page copy
- Creating email sequences
- Drafting social media content
- Writing product documentation
- Creating ad copy

## Expected Outputs

Content saved to `docs/marketing/`:
- `landing-page.md` - Main landing page copy
- `email-sequences/` - Email templates by sequence type
- `social-posts/` - Platform-organized posts
- `ad-copy/` - Advertising templates

## Quality Criteria

Content must include:
- Multiple headline variations (3+) for testing
- Clear calls-to-action
- Benefit-focused messaging
- Meta descriptions for SEO
- Short, scannable paragraphs
