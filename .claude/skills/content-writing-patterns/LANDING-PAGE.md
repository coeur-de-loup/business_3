# Landing Page Copy Framework

## Above the Fold

### Headline (3 variations minimum)

**Formula 1: Result + Timeframe**
```
[Achieve X] in [Y time] — without [common obstacle]
```

**Formula 2: Pain → Solution**
```
Stop [pain]. Start [desired state].
```

**Formula 3: Audience + Result**
```
[Result] for [specific audience] who [situation]
```

### Subheadline
Expand on headline. Include:
- What it is (briefly)
- Who it's for
- Key differentiator

### CTA Button
Action-oriented, first person:
- "Start my free trial"
- "Get my [result]"
- "Join 2,847 [audience type]"

## Problem Section

### Structure
```markdown
## [Relatable situation heading]

You know the feeling...

[Describe their current painful situation vividly]

And it gets worse:
- [Consequence 1]
- [Consequence 2]
- [Consequence 3]

There has to be a better way.
```

## Solution Section

### Structure
```markdown
## Introducing [Product Name]

[One sentence: what it is and core benefit]

### Here's how it works:

**Step 1: [Action]**
[Brief description + outcome]

**Step 2: [Action]**
[Brief description + outcome]

**Step 3: [Action]**
[Brief description + outcome]

It's that simple.
```

## Features → Benefits

### Format
```markdown
| Feature | Benefit |
|---------|---------|
| [What it has] | [What that means for them] |
| Automated reports | Save 5 hours every week |
| One-click integration | Set up in 2 minutes, not 2 days |
| Real-time analytics | Make decisions before it's too late |
```

## Social Proof

### Testimonial Format
```markdown
> "[Specific result achieved] — [Product] changed how we [do thing]."
>
> **[Name], [Title] at [Company]**
> [Metric: "Saved $X/month" or "Grew Y%"]
```

### Stats Bar
```
[Number]+ [Users/Companies/Projects]
[Number]% [Improvement metric]
[Number] [Time saved/Revenue generated]
```

## FAQ Section

Address objections as questions:
1. "Is this right for [their situation]?"
2. "How long does it take to [get result]?"
3. "What if [common fear]?"
4. "How is this different from [alternative]?"
5. "What's included in [pricing tier]?"

## Final CTA

```markdown
## Ready to [achieve result]?

[Restate core benefit in one sentence]

[CTA Button: "Start free trial"]

No credit card required. Cancel anytime.
```
